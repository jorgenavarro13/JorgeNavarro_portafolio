# Este es mi portafolio web

Este proyecto es un portafolio web que muestra mis habilidades, proyectos y experiencia profesional. Está diseñado para ser una representación visual de mi trabajo y para facilitar la conexión con posibles empleadores o colaboradores.

Link del proyecto: [Mi Portafolio Web](https://jorgenavarro13.github.io/JorgeNavarro_portafolio/)

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
